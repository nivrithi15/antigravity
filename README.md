# 🌌 Voxara – Futuristic Election Assistant

**Voxara** is an interactive, AI-powered web app designed to help users understand the election process in a simple, engaging, and visually immersive way.

With a futuristic **aurora-inspired UI** and dynamic country-based theming, Voxara makes learning about elections intuitive and enjoyable.

---

## 🚀 Live Demo
https://voxaraa.netlify.app/


---

## 🧠 Features

### ✨ Interactive AI Chatbot

* Guides users through the election process step-by-step
* Simple, beginner-friendly explanations
* Conversational UI with clickable actions

### 🗳️ Election Timeline

* Visual breakdown of:

  * Election announcement
  * Candidate nominations
  * Campaigning
  * Voting day
  * Vote counting
  * Results

### 🎯 Quiz Mode

* Quick multiple-choice questions
* Instant feedback
* Reinforces learning

### 🌍 Dynamic Country-Based Theming

* Select your country
* UI adapts to flag colors
* Icons, buttons, and gradients update automatically

### 🌌 Aurora UI Design

* Smooth, animated gradient background (northern lights inspired)
* Glassmorphism cards
* Soft glow + modern UI elements

---

## 🖥️ Tech Stack

* Frontend: HTML / CSS / JavaScript 
* UI Design: Antigravity prototype
* Deployment: Netlify

## 🎨 Design Philosophy

Voxara focuses on:

* Simplicity → Easy for first-time voters
* Engagement → Interactive learning
* Visual Appeal → Futuristic aurora UI
* Accessibility → Clean, readable interface
